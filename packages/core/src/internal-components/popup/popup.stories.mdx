## Installation

To use the internal component add the register import to the `register.ts` of
the consuming component.

```typescript
import '@cds/core/internal-components/popup/register.js';
```

```html
<cds-internal-popup>
  <div>
    <h1>I am a popup</h1>
    <p>Hello, I am a popup. You can put whatever content you want in me.</p>
  </div>
</cds-internal-popup>
```
